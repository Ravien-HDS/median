  • Test je capaciteiten

Doe onze capaciteitstest en sta sterker in je schoenen op de arbeidsmarkt

Ben je al een tijdje op zoek naar een job, of zou je graag bewijzen dat je een hogere functie aankan? Dan is onze test hét document bij uitstek dat jij kan gebruiken om te bewijzen wat je waard bent!

Hoe gaat onze test in zijn werk?

Je contacteert ons voor een afspraak. Dan nodigen wij je uit bij ons op kantoor voor een testmoment. Reken op twee à drie uur om deze te voltooien.

De test wordt ingevuld aan de computer, waarbij je gebruik mag maken van pen, papier en een rekenmachine. Ze bestaat uit vier delen:

  • Verbale vaardigheden
  • Numerieke vaardigheden
  • Abstract denken
  • Persoonlijkheidstest

De waarde van onze test

Al onze testen werken volgens de gestandaardiseerde methode van 'The Bridge'. Deze internationale standaard zorgt ervoor dat je test algemeen erkend wordt. Zo kan een potentiële werkgever jouw capaciteiten en persoonlijkheid beoordelen en je een job op jouw niveau aanbieden. Jij kan zo makkelijk bewijzen wat je waard bent!
